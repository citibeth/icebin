Introduction

GLINT2 is a library intended for the coupling of climate models (GCMs) and ice models. It was developed by Robert Fischer (@citibob) at NASA GISS in New York City, and is licensed via the GPL v3. GLINT2 is written in C++, and is intended for use with GCMs and climate models written in C/C++ and Fortran 2003.

GLINT2 implements the transformations described in the paper A System of Conservative Regridding for Ice / Atmosphere Coupling in a GCM (not yet published).

GLINT2 code may be downloaded at this GitHub repository. GLINT2 documentation is found at this project's Wiki.

If you are interested in using GLINT2 or joining as a committer, please email Robert Fischer robert.fischer@nasa.gov
