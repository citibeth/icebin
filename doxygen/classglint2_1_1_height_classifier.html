Detailed Description

Closure used to determine the elevation class of each GCM grid cell, based on its elevation.

This is built using a simple STL collection class, plus standard STL binary search subroutines. This class is set up to work with an unusual array structure (see HeightMaxArray).

Typically, this class is used as a temporary decorator to a HeightMaxArray instance.
