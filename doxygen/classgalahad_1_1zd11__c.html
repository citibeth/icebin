Detailed Description

C++ Peer class for the Fortran hsl_zd11_double::zd11_type.

Elements of the Fortran derived type may be safely accessed via the references (scalars) and pointers (arrays) in this class. See hsl_zd11_double::zd11_type for descriptions of these elements. NOTE: This class must always match the Fortran derived type zd11_c.

See Also
hsl_zd11_double::zd11_type, hsl_zd11_double_x::zd11_c
