virtual void giss::SparseMatrix::set ( int  row,
int  col,
double const  val,
DuplicatePolicy  dups = DuplicatePolicy::REPLACE 
)
pure virtual

Set the value of an element in the matrix.

Checks that row and column indices are within range. If the matrix is triangular, swaps row and col if needed, to get the element in the correct upper/lower triangle.

Parameters
rowRow of element to set (base=0).
colColumn of element to set (base=0).
valValue to set at (row, col).
dupsWhat should be done if there was an existing element in (row,col)?
dups=DuplicatePolicy::REPLACE
Replace with new value.
dups=DuplicatePolicy::ADD
Add new value to existing value.
Note that only MapSparseMatrix supports these modes. VectorSparseMatrix and ZD11SparseMatrix ignore the values of dups.
