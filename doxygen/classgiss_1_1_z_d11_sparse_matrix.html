Detailed Description

A SparseMatrix based on external Fortran HSL_ZD11 storage.

A standardized sparse matrix interface to the Fortran ZD11 data structure. ZD11SparseMatrix wraps an existing ZD11 data structure — which itself is a peer to an underlying hsl_zd11_double::zd11_type Fortran derived type.

Use this class when you need to construct a ZD11-type matrix and pass it to/from a Fortran subroutine. NOTE: Not only the dimensions, but also the number of non-zero elements must be pre-determined. This data structure is also used to copy from an existing SparseMatrix of another type.

See Also
hsl_zd11_double::zd11_type, ZD11, hsl_zd11_double_x::hsl_zd11_c_init, copy
