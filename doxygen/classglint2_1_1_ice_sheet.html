Detailed Description

NOTE: The Interpolation Grid (grid4) is the grid to which we

interpolate, and which ultimately defines the basis functions in elevation point space.

By default, this is the Ice Grid. But for L0 ice grids, we can use the Exchange Grid as the Interpolation Grid, in order to keep the RM matrix local.
