Convert a Numpy array to a blitz one, using the original's data (no copy).

N is the rank of the array. This should be called AFTER appropriate typechecking has taken place.

See Also
: http://mail.scipy.org/pipermail/numpy-discussion/2004-March/002892.html

Use this template to convert from PyObject to a blitz::Array of a specific type and dimensions.

Example: PyObject *Val_py; auto Val(py_to_blitz<double,3>(Val_py, "Val", {3, -1, 4}));
